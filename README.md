> A rainbow of patterns

**Saturated Sands** expands upon the faded terracotta in mirage, providing a miriad of colours to build murals, flooring or add extra detail and shading to any build.

### ✨ Features

* **Patterned Terracotta:** Dye your faded terracotta with any dye colou.r

* **Stonecutting Recipes:** Change between any pattern of terracotta using the stonecutter!

* **Fully Configurable:** Enable or disable any feature within the mod through the config, including disabling the stonecutter recipes or changing what can be seen in the creative tab!

* **More coming soon:** This mod is still in active development and is likely to add more patterns and more mod compatibility, feel free to suggest new features on the discord!
